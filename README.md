# early-warning
Multi-Source Early Warning Dashboard project — focused on Influenza, Malaria, Cholera, and zika using Google Trends fused with official data for predictions, anomaly alerts, visualizations, and a public API
# Infodemiology Early Warning System (EWS)

An early warning system for infectious disease surveillance using
Google Trends data.

## Tech Stack
- Backend: FastAPI, SQLAlchemy, SQLite
- Frontend: React, TypeScript, Vite, Tailwind
- Data: Google Trends (Interest Over Time)

## Features
- CSV ingestion of Google Trends data
- Multi-disease support
- Dataset registry
- Interactive dashboard (in progress)
